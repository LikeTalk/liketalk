각각 함수모듈 소개

debug.py
: 디버깅을 위해서 이렇게 저렇게 해보려는 테스트모듈

test.py
: 테스트모듈

admin.py
: DB에 접속해서 DB 내용을 채우거나 지우는 기능을 하는 모듈
: 함수들의 Return 값이 없기 때문에, 서버 오류 페이지를 뱉어낼 것이다 .

candidate.py
: 개별 인원들의 얼굴을 보여준다.
: 나중에 팀을 위한 홈페이지를 새롭게 만들 때 좋은 역할을 할 것으로 보인다.

view_page.py
: html 페이지를 지정해서 뿌려주는 함수

what_match
: 이상형 월드컵의 핵심 모듈
: 어떤 게임을 할지 계산해서 뿌려주고
: 투표를 받아서 기록하는 모듈

user_account
: 회원가입, 로그인, 로그아웃을 담당하는 모듈
